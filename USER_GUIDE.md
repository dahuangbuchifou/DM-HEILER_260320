## 📋 详细使用步骤

### 方法 1：手动打开演出页面（推荐 ✅）

**步骤：**

1. **打开大麦 App**
   ```
   在手机桌面找到"大麦"图标并打开
   ```

2. **搜索演出**
   ```
   在大麦 App 中搜索：刘宇宁
   点击搜索结果中的演出卡片
   ```

3. **确认页面**
   ```
   确保页面底部有"预约抢票"或"立即购买"按钮
   页面显示演出详情（时间、场馆、票价等）
   ```

4. **启动助手**
   ```
   返回助手 App
   点击"智能抢票（推荐）"
   填写信息（歌手、日期、价位）
   点击"保存任务"
   ```

5. **自动完成**
   ```
   助手会自动：
   ✅ 检测当前页面
   ✅ 点击"预约抢票"按钮
   ✅ 选择场次
   ✅ 选择票档
   ✅ 确认预约
   ✅ 处理弹窗
   ```

## ⚠️ 常见问题

### Q1: 助手提示"未找到预约抢票按钮"
**原因：** 不在演出详情页  
**解决：** 手动打开演出详情页后再启动助手

### Q2: 助手提示"未找到票档"
**原因：** 
- 票档已售罄
- 票档文本格式不匹配

**解决：**
- 检查票档是否还有余票
- 尝试其他价位

### Q3: 助手提示"未找到确认按钮"
**原因：** 页面还未加载完成  
**解决：** 等待页面加载完成后再启动助手

### Q4: 助手卡在某个步骤
**原因：** 页面元素变化或网络延迟  
**解决：**
- 检查网络连接
- 重新打开演出页面
- 重启助手

助手支持以下票档格式：

- `¥1280` - 带人民币符号
- `1280 元` - 带单位
- `内场 1280` - 带区域
- `看台 480` - 带区域
