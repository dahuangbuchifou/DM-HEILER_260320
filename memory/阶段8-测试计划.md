## 🚨 异常处理

**常见问题：**

1. **搜索不到演出**
   - 检查关键词是否正确
   - 检查演出是否已开售/下架

2. **票档选择失败**
   - 检查票档是否售罄
   - 检查 `selectedPrice` 格式是否匹配

3. **观众选择失败**
   - 检查大麦 App 是否已添加观演人
   - 检查 `audienceName` 是否匹配

4. **无法到达支付页**
   - 检查是否有限购
   - 检查是否有答题/验证码环节
