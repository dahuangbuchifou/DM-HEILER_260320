## 🔍 详细方案分析

### 方案 1：GPT-SoVITS（本地电脑 API + 内网穿透）⭐⭐⭐⭐⭐

**架构：**
```
OpenClaw 服务器 → Cloudflare Tunnel → 本地电脑 → GPT-SoVITS API
```

**优势：**
- ✅ 效果极佳（中文最自然）
- ✅ 完全免费
- ✅ 已训练好的模型可直接用
- ✅ 支持情感控制
- ✅ 只需 1 分钟声音样本

**劣势：**
- ⚠️ 本地电脑需要开机
- ⚠️ 依赖家庭网络稳定性
- ⚠️ 需要配置内网穿透

**部署步骤：**
```bash
# 1. 本地电脑启动 API
cd /path/to/GPT-SoVITS
python api.py --host 0.0.0.0 --port 9880

# 2. 安装 Cloudflare Tunnel
npm install -g @cloudflare/cloudflared

# 3. 创建隧道
cloudflared tunnel --url http://localhost:9880

# 4. 获取公网地址（类似：https://xxx-xxx-xxx.cfargotunnel.com）
# 5. 在 OpenClaw 配置 API 地址
```

**API 调用示例：**
```bash
curl -X POST "https://xxx-xxx-xxx.cfargotunnel.com" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "你好，我是婉儿",
    "text_language": "zh",
    "refer_wav_path": "/path/to/sample.wav",
    "prompt_text": "声音样本的原文"
  }' --output output.wav
```

**成本：** 免费  
**延迟：** 1-3 秒（取决于网络）  
**维护：** 低（本地电脑常开即可）

### 方案 2：GPT-SoVITS（服务器 CPU 推理）⭐⭐⭐

**架构：**
```
OpenClaw 服务器 → 本地 GPT-SoVITS（CPU 模式）
```

**优势：**
- ✅ 完全免费
- ✅ 无需外部依赖
- ✅ 数据本地，隐私好
- ✅ 已训练模型可直接用

**劣势：**
- ❌ 推理速度慢（10-30 秒/句）
- ❌ 占用服务器 CPU 资源
- ❌ 长文本体验差

**部署步骤：**
```bash
# 1. 克隆项目
git clone https://github.com/RVC-Boss/GPT-SoVITS.git
cd GPT-SoVITS

# 2. 安装依赖
pip install -r requirements.txt

# 3. 下载预训练模型
# HuggingFace: https://huggingface.co/lj1995/GPT-SoVITS

# 4. CPU 模式启动
python api.py --device cpu --host 127.0.0.1 --port 9880
```

**成本：** 免费  
**延迟：** 10-30 秒/句  
**维护：** 低

---

### 方案 3：ElevenLabs（云端 API）⭐⭐⭐⭐

**架构：**
```
OpenClaw 服务器 → ElevenLabs API（海外）
```

**优势：**
- ✅ 效果极佳（业界顶尖）
- ✅ 部署极简（API 调用）
- ✅ 支持声音克隆（1 分钟样本）
- ✅ 多语言支持好
- ✅ 情感丰富

**劣势：**
- ❌ 需要科学上网（服务器访问海外）
- ❌ 免费版额度有限（1 万字符/月）
- ❌ 付费版 $5/月起
- ❌ 中文效果略逊于 GPT-SoVITS

**定价：**
| 套餐 | 价格 | 字符额度 | 功能 |
|------|------|----------|------|
| Free | $0 | 1 万/月 | 基础克隆 |
| Starter | $5 | 3 万/月 | 商业使用 |
| Creator | $22 | 10 万/月 | 高级功能 |

**部署步骤：**
```bash
# 1. 注册账号
https://elevenlabs.io

# 2. VoiceLab → Add Generative or Cloned Voice
# 3. 上传 1-5 分钟声音样本
# 4. 获取 API Key

# 5. OpenClaw 配置
export ELEVENLABS_API_KEY="your_key"
```

**API 调用示例：**
```bash
curl -X POST "https://api.elevenlabs.io/v1/text-to-speech/{voice_id}" \
  -H "xi-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "你好，我是婉儿",
    "model_id": "eleven_multilingual_v2",
    "voice_settings": {"stability": 0.5, "similarity_boost": 0.75}
  }' --output output.mp3
```

**成本：** $0-5/月  
**延迟：** 1-2 秒  
**维护：** 极低
