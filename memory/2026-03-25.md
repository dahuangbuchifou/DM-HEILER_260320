## 🎯 学到的教训

### 1. Kotlin 版本一致性
- AGP 8.7.0 + Kotlin 1.9.25 需要 stdlib 版本一致
- 版本不匹配会导致 Android SDK 常量无法解析

### 2. 枚举 vs 字符串
- Kotlin 强类型，枚举和字符串不能互换
- when 语句分支类型必须与表达式一致

### 3. Kotlin object 单例
- `object` 是单例，不能实例化
- 直接调用方法：`AccessibilityUtils.isServiceEnabled()`

### 4. Android SDK 常量
- 某些常量可能因版本问题无法解析
- 直接使用整数值更可靠（`GLOBAL_ACTION_BACK` = 1）

### 5. 功能设计 vs Bug
- "同步失败"提示是正常功能设计，不是 bug
- 需要在大麦 App 详情页才能抓取信息
