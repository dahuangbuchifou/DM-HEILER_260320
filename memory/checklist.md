## 自动交互按钮

检测到信息不完整时，自动点击：
- [ ] 预约抢票
- [ ] 确定
- [ ] 立即提交
- [ ] 下一步
- [ ] 确认选座
- [ ] 配送方式
- [ ] 提交订单

- 微信支付

- 付款金额确认
