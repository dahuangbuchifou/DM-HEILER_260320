# 2026-03-31 - DM-HEILER 重大功能修复

## 🐛 Bug 修复

### 删除功能失效
- **问题：** 任务列表删除按钮点击后只显示 Toast，没有真正删除
- **原因：** MainActivity.kt 中 `setupTaskList()` 的 `onDelete` 回调未连接实际删除函数
- **修复：** 将 `onDelete = { Toast... }` 改为 `onDelete = { deleteTask(it) }`
- **版本：** v2.2.1 (08:15)
