# 🎫 大麦信息抓取增强版 - 快速参考

---

## 📋 新增功能

✅ **完整信息抓取**
- 演出标题、时间（多场次）、地点、票价（所有档位）、库存状态

✅ **自动交互流程**
- 预约抢票、确定、立即提交、下一步、确认选座、配送方式、提交订单

✅ **付款界面检测**
- 检测到支付宝/微信/银行卡支付等关键词自动停止

✅ **详细日志报告**
- 输出完整演唱会信息报告

---

## 🚀 快速使用

```kotlin
// 1. 创建增强版控制器
val enhancedController = TicketGrabbingControllerEnhanced(
    accessibilityService = this,
    screenCapture = screenCaptureService,
    analyzer = screenAnalyzer
)

// 2. 创建任务
val task = TicketTask(
    concertKeyword = "【烟台】李宗盛 2026『有歌之年』巡回演唱会",
    grabDate = "2026-03-30",
    grabTime = 1711785600000L,
    sessionId = "2026-05-15-19:30",
    priceTiers = "1580,1280,880",
    selectedPrice = "1580",
    audienceName = "张三",
    audienceIndex = 1,
    grabMode = "snap"
)

// 3. 开始抢票
enhancedController.startGrabbing(task)
```

## 🔄 工作流程

```
1. 检测分屏 → 2. 提取信息 → 3. 检查完整性
→ 4. 自动导航（如需要） → 5. 选择票档 → 6. 选择观众
→ 7. 提交订单 → 8. 到达付款界面 → 停止
```

## ⚙️ 配置

```kotlin
// 自动交互按钮
ACTION_BUTTONS = ["预约抢票", "确定", "立即提交", "下一步", 
                  "确认选座", "配送方式", "提交订单", ...]

// 付款界面检测
PAYMENT_KEYWORDS = ["支付宝支付", "微信支付", "银行卡支付", 
                    "付款金额", "确认支付", ...]

// 截屏帧率
screenCapture.startCapture(fps = 2)  // 2FPS
```
