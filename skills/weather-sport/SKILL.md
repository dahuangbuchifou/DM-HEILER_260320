# 🌤️ 天气 + 运动建议 Skill

> 根据天气、温度、空气质量推荐运动方案

---

##  功能

1. **天气获取**
   - 当前天气（温度、湿度、风力）
   - 短期预报（今日/明日）
   - 特殊天气预警

2. **运动推荐**
   - 根据天气推荐户外运动/室内运动
   - 根据温度推荐运动强度
   - 根据空气质量建议

3. **穿衣建议**
   - 根据温度推荐穿衣
   - 特殊天气提醒（雨具、防晒）

---

## 🎯 推荐规则

### 户外运动条件

| 条件 | 推荐 | 不推荐 |
|------|------|--------|
| 温度 | 10-28°C | <5°C 或 >30°C |
| 降水 | 无雨/雪 | 雨/雪 |
| 风力 | <3 级 | ≥5 级 |
| 空气质量 | 优/良 | 重度污染 |

### 运动推荐

| 天气 | 推荐运动 | 备注 |
|------|----------|------|
| ☀️ 晴朗 | 跑步、骑行、羽毛球 | 注意防晒 |
| ⛅ 多云 | 跑步、健身、爬山 | 舒适 |
| 🌧️ 小雨 | 室内健身、瑜伽 | 带伞 |
| 🌧️ 大雨 | 室内运动 | 避免外出 |
| ❄️ 小雪 | 室内为主 | 注意保暖 |
| 🌨️ 大雪 | 室内运动 | 避免外出 |
| 🌫️ 雾霾 | 室内运动 | 开空气净化器 |

## 🔧 技术实现

### 天气 API

```bash
# wttr.in（无需 API Key）
curl "wttr.in/大连?format=3"
curl "wttr.in/大连?format=j1"  # JSON

# Open-Meteo（免费，需 HTTP 请求）
curl "https://api.open-meteo.com/v1/forecast?latitude=38.9&longitude=121.6&current_weather=true"
```

### 运动推荐逻辑

```python
def recommend_activity(weather, temp, aqi):
    if weather in ['rain', 'snow'] or temp < 5 or temp > 30:
        return "室内运动"
    if aqi > 150:
        return "室内运动（空气质量差）"
    if temp >= 10 and temp <= 28:
        return "户外运动（舒适）"
    return "适度户外活动"
```
