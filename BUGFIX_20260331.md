### 1. 删除功能修复 ✅

**问题原因：**
- `MainActivity.kt` 中 `setupTaskList()` 方法的 `onDelete` 回调只显示 Toast 提示
- 没有调用实际的 `deleteTask()` 函数

**修复方案：**
```kotlin
// 修复前
private fun setupTaskList() {
    taskAdapter = TaskAdapter(
        onEdit = { Toast.makeText(this, "编辑：${it.name}", Toast.LENGTH_SHORT).show() },
        onDelete = { Toast.makeText(this, "删除：${it.name}", Toast.LENGTH_SHORT).show() },
        onStartStop = { handleStartStopTask(it) }
    )
}

// 修复后
private fun setupTaskList() {
    taskAdapter = TaskAdapter(
        onEdit = { showEditTaskDialog(it) },
        onDelete = { deleteTask(it) },
        onStartStop = { handleStartStopTask(it) }
    )
}
```

**涉及文件：**
- `app/src/main/java/com/damaihelper/ui/MainActivity.kt` (第 113-114 行)

### 2. 智能抢票自动跳转增强 ✅

**问题原因：**
- `SmartGrabActivity` 创建任务后直接 `finish()` 返回主页
- 没有自动启动抢票服务和打开大麦 App

**修复方案：**
在 `saveTask()` 方法中添加自动启动逻辑：
```kotlin
val taskId = db.taskDao().insertTask(task).toInt()

Toast.makeText(this@SmartGrabActivity, "✅ 任务创建成功！即将自动跳转到大麦...", Toast.LENGTH_LONG).show()

// 延迟 1 秒后启动抢票服务并打开大麦
android.os.Handler(android.os.Looper.getMainLooper()).postDelayed({
    try {
        // 启动抢票服务
        val service = TicketGrabbingAccessibilityService.getInstance()
        if (service != null) {
            // 自动开始抢票流程
            service.startGrabbing(task)
            
            // 打开大麦 App
            val damaiIntent = Intent(Intent.ACTION_VIEW).apply {
                setPackageName("cn.damai")
                setClassName("cn.damai", "cn.damai.main.activity.MainActivity")
                addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
            }
            
            // 尝试启动大麦
            try {
                startActivity(damaiIntent)
                Toast.makeText(this@SmartGrabActivity, "🎫 已启动大麦 App，正在自动搜索...", Toast.LENGTH_LONG).show()
            } catch (e: Exception) {
                // 如果直接启动失败，尝试通用方式
                val webIntent = Intent(Intent.ACTION_VIEW).apply {
                    data = Uri.parse("https://www.damai.cn/")
                    addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
                }
                startActivity(webIntent)
                Toast.makeText(this@SmartGrabActivity, "🌐 已打开大麦网页版，请手动操作", Toast.LENGTH_LONG).show()
            }
        } else {
            Toast.makeText(this@SmartGrabActivity, "⚠️ 无障碍服务未启动，请手动开启", Toast.LENGTH_LONG).show()
        }
    } catch (e: Exception) {
        Toast.makeText(this@SmartGrabActivity, "⚠️ 自动启动失败，请手动操作", Toast.LENGTH_LONG).show()
    }
}, 1000)

finish()
```

**自动流程：**
1. ✅ 创建任务并保存到数据库
2. ✅ 启动 `TicketGrabbingAccessibilityService.startGrabbing(task)`
3. ✅ 打开大麦 App（优先原生，失败则网页版）
4. ✅ 抢票服务自动执行：打开大麦 → 搜索演出 → 选择票档 → 选择观众 → 提交订单

**涉及文件：**
- `app/src/main/java/com/damaihelper/ui/SmartGrabActivity.kt` (第 245-285 行)
