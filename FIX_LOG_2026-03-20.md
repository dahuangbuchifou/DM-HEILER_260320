## 🔍 发现的问题及修复

### 1. Android 14+ 前台服务权限

**问题：** Android 14 (API 34) 要求前台服务必须声明 `specialUse` 属性

**修复：**
```xml
<!-- 已确认配置正确 -->
<uses-permission android:name="android.permission.FOREGROUND_SERVICE_SPECIAL_USE" />
```

**新增权限：**
```xml
<!-- 防止抢票时 CPU 休眠 -->
<uses-permission android:name="android.permission.WAKE_LOCK" />

<!-- 开机自启（可选） -->
<uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED" />
```

---

### 2. 时间类型安全问题

**问题：** `TicketTask.grabTime` 可能存在类型转换错误

**修复：** 确认使用 `Long` 类型，在 `TicketGrabbingAccessibilityService.kt` 中添加安全转换：

```kotlin
val grabTimeMillis = if (task.grabTime is Long) {
    task.grabTime
} else {
    task.grabTime.toString().toLongOrNull()
        ?: throw IllegalArgumentException("抢票时间格式错误：${task.grabTime}")
}
```

---

### 3. 页面检测容错

**问题：** 演出信息抓取时页面检测过于严格

**修复：** 在 `TicketGrabbingAccessibilityService.kt` 中优化 `isOnDamaiDetailPage()`：

## 📝 使用说明

### 抢票流程：

1. **启用无障碍服务**
   - 设置 → 无障碍 → 已下载的服务 → 开启"抢票助手"

2. **抓取演出信息（可选）**
   - 打开大麦 App → 进入演出详情页
   - 返回抢票助手 → 点击"📥 从大麦抓取演出信息"

3. **创建抢票任务**
   - 点击"➕ 手动添加任务"
   - 填写演出名称、日期、票价等信息
   - 保存任务

4. **开始抢票**
   - 在任务列表中点击任务
   - 确保提前进入大麦 App
   - 服务会自动执行抢票流程
