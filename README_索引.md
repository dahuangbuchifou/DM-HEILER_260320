## 📌 项目概述

**DM-HEILER** 是一个智能票务抢购辅助系统，专为大麦网等票务平台设计。

### 核心功能
- ✅ 自动监控演出/比赛票务信息
- ✅ 智能反反爬虫策略
- ✅ 自动跳转付款界面
- ✅ 多账号/多设备支持
- ✅ 实时状态推送
