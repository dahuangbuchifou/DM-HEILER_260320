# 关键参数配置 - v1.0.0

## API 抢票参数

### 超时控制

| 参数 | 值 | 说明 |
|------|-----|------|
| `GRAB_TIMEOUT_MS` | 3000ms | 单次抢票超时 |
| `connectTimeout` | 30s | HTTP 连接超时 |
| `readTimeout` | 30s | HTTP 读取超时 |
| `writeTimeout` | 30s | HTTP 写入超时 |

### 重试策略

| 参数 | 值 | 说明 |
|------|-----|------|
| `MAX_RETRY_COUNT` | 3 | 最大重试次数 |
| `RETRY_INTERVAL_MS` | 100ms | 基础重试间隔 |
| 退避算法 | 线性 | 每次增加 100ms |

### 熔断机制

| 参数 | 值 | 说明 |
|------|-----|------|
| `CONSECUTIVE_FAILURES_THRESHOLD` | 5 | 连续失败阈值 |
| 熔断后动作 | 切换 UI 策略 | 自动降级 |

## 反检测参数

### 指纹轮换

| 参数 | 值 | 说明 |
|------|-----|------|
| `FINGERPRINT_ROTATION_INTERVAL_MS` | 300,000ms (5 分钟) | 指纹轮换间隔 |
| 轮换内容 | User-Agent, DeviceID, Platform | 设备特征 |

### 频率限制

| 参数 | 值 | 说明 |
|------|-----|------|
| `MIN_REQUEST_INTERVAL_MS` | 50ms | 最小请求间隔 |
| `MAX_REQUEST_INTERVAL_MS` | 200ms | 最大请求间隔 |
| 随机因子 | SecureRandom | 加密级随机 |

### TLS 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| TLS 版本 | TLS 1.2, TLS 1.3 | 支持版本 |
| HTTP 版本 | HTTP/1.1 | 禁用 HTTP/2 |
| 连接池 | 5 连接，5 分钟 | 复用配置 |
| 重试 | false | 禁用自动重试 |

## 时间同步参数

| 参数 | 值 | 说明 |
|------|-----|------|
| NTP 服务器 | pool.ntp.org | 时间源 |
| 同步间隔 | 每次启动 | 同步频率 |
| 抢票提前量 | 200-500ms | 提前开始 |
