### 3. 核心功能

#### 3.1 完整信息提取

```kotlin
// 提取演出标题（支持【】、「」等格式）
extractConcertTitle(textBlocks)

// 提取多场次时间
extractShowTimes(textBlocks)

// 提取演出地点
extractLocation(textBlocks)

// 提取所有票价档位
extractAllPriceTiers(textBlocks)

// 提取库存状态
extractStockStatus(textBlocks)
```

**识别模式：**
- 标题：`【[^】]*】[^\n]{10,}`、`「[^」]*」[^\n]{10,}`
- 时间：`(\d{4}-\d{2}-\d{2})[^\n]*(周[五六日一二三四])[^\n]*(\d{2}:\d{2})`
- 地点：体育中心、体育馆、会展中心、大剧院、体育场
- 价格：`(¥|￥)?\s*(\d+)(元)?`

#### 3.2 信息完整性检查

```kotlin
private fun checkInfoCompleteness(info: CompleteConcertInfo): Boolean {
    return info.title.isNotEmpty() && 
           info.showTimes.isNotEmpty() && 
           info.location.isNotEmpty() && 
           info.priceTiers.isNotEmpty()
}
```

**工作流程：**
1. 截屏识别当前页面
2. 查找可点击按钮
3. 点击并等待页面加载
4. 重新检查信息完整性
5. 循环直到信息完整或到达付款界面

#### 3.4 付款界面检测

```kotlin
PAYMENT_KEYWORDS = [
    "支付宝支付", "微信支付", "银行卡支付",
    "付款金额", "确认支付", "立即支付", "选择支付方式"
]

private fun isPaymentPage(textBlocks: List<TextBlock>): Boolean {
    val allText = textBlocks.joinToString(" ") { it.text }
    return PAYMENT_KEYWORDS.any { allText.contains(it, ignoreCase = true) }
}
```

## 🚀 使用方法

### 快速开始

```kotlin
// 1. 创建控制器
val controller = TicketGrabbingControllerEnhanced(
    accessibilityService = this,
    screenCapture = screenCaptureService,
    analyzer = screenAnalyzer
)

// 2. 创建任务
val task = TicketTask(
    concertKeyword = "李宗盛 2026 巡演",
    grabDate = "2026-03-30",
    grabTime = 1711785600000L,
    priceTiers = "1580,1280,880",
    selectedPrice = "1580",
    audienceName = "张三"
)

// 3. 开始抢票
controller.startGrabbing(task)
```

### 状态监听

```kotlin
controller.stateFlow.collect { state ->
    when (state) {
        GrabbingState.INFO_EXTRACTING -> println("📋 提取信息...")
        GrabbingState.PRICE_SELECTING -> println("🎫 选票档...")
        GrabbingState.AUDIENCE_SELECTING -> println("👤 选观众...")
        GrabbingState.SUBMITTING -> println("📝 提交订单...")
        GrabbingState.PAYMENT_PAGE -> println("💳 到达付款界面！")
        GrabbingState.COMPLETED -> println("✅ 完成！")
        else -> {}
    }
}
```

## ⚠️ 注意事项

### 必需条件
1. ✅ Android 10+（支持 MediaProjection API）
2. ✅ 分屏模式（助手 + 大麦）
3. ✅ 截屏权限授权
4. ✅ 无障碍服务权限

### 使用建议
1. 确保大麦页面信息完整显示
2. 保持网络畅通（OCR 识别）
3. 观察日志输出确认状态
4. 到达付款界面后手动完成支付

### 错误处理
- 信息不完整：自动导航获取
- 识别失败：多次重试（最多 10 次）
- 按钮未找到：尝试备选按钮
- 超时：10 秒超时保护
