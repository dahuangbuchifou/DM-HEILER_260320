## 🚀 快速开始

### 1. 启用增强版控制器

在 `MainActivity.kt` 或相关服务中：

```kotlin
// 创建增强版控制器
val enhancedController = TicketGrabbingControllerEnhanced(
    accessibilityService = this,
    screenCapture = screenCaptureService,
    analyzer = screenAnalyzer
)

// 开始抢票
enhancedController.startGrabbing(task)
```

### 2. 创建抢票任务

```kotlin
val task = TicketTask(
    concertKeyword = "【烟台】李宗盛 2026『有歌之年』巡回演唱会",
    grabDate = "2026-03-30",
    grabTime = 1711785600000L,
    sessionId = "2026-05-15-19:30",  // 指定场次
    priceTiers = "1580,1280,880",     // 多票档备选（从高到低）
    selectedPrice = "1580",           // 首选票档
    audienceName = "张三",
    audienceIndex = 1,                // 默认第一个观众
    grabMode = "snap"                 // 秒杀模式
)
```

### 3. 开始抢票

```kotlin
// 确保已分屏（助手 + 大麦）
// 点击"开始抢票"按钮

// 观察日志输出
enhancedController.stateFlow.collect { state ->
    when (state) {
        GrabbingState.INFO_EXTRACTING -> println("📋 提取演出信息...")
        GrabbingState.PRICE_SELECTING -> println("🎫 选择票档...")
        GrabbingState.AUDIENCE_SELECTING -> println("👤 选择观众...")
        GrabbingState.SUBMITTING -> println("📝 提交订单...")
        GrabbingState.PAYMENT_PAGE -> println("💳 到达付款界面！")
        GrabbingState.COMPLETED -> println("✅ 抢票完成！")
        GrabbingState.FAILED -> println("❌ 抢票失败")
        else -> {}
    }
}
```

## 💳 付款界面检测

### 检测关键词

```kotlin
private val PAYMENT_KEYWORDS = listOf(
    "支付宝支付",
    "微信支付", 
    "银行卡支付",
    "付款金额",
    "确认支付",
    "立即支付",
    "选择支付方式"
)
```

### 检测逻辑

```kotlin
private fun isPaymentPage(textBlocks: List<TextBlock>): Boolean {
    val allText = textBlocks.joinToString(" ") { it.text }
    
    for (keyword in PAYMENT_KEYWORDS) {
        if (allText.contains(keyword, ignoreCase = true)) {
            Log.i(TAG, "✅ 检测到付款界面关键词：$keyword")
            return true
        }
    }
    
    return false
}
```

## 🐛 常见问题

### Q1: 信息提取不完整怎么办？

**A:** 增强版会自动尝试以下方法：
1. 多次截屏识别（最多 10 次）
2. 自动点击按钮导航（预约、确定、下一步等）
3. 如果仍然不完整，会输出已提取的信息并继续流程

**手动解决：**
- 确保大麦页面信息完整显示
- 检查网络连接
- 重启分屏模式

### Q2: 自动点击不生效？

**A:** 可能原因：
- 无障碍服务未开启
- 分屏模式未正确检测
- 按钮文字识别失败

**解决：**
- 检查无障碍服务权限
- 重新分屏
- 查看日志确认识别结果

### Q3: 如何知道到达了付款界面？

**A:** 观察日志输出：
```
✅ 检测到付款界面关键词：支付宝支付
✅ 到达付款界面，停止自动操作
```

状态会变为 `PAYMENT_PAGE`。

### Q4: 多票档备选如何使用？

**A:** 在任务中配置 `priceTiers`：
```kotlin
priceTiers = "1580,1280,880"  // 从高到低排列
```

系统会按顺序尝试每个票档，直到找到有票的。
