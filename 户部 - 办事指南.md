_版本：v1.0_  
_更新日期：2026-04-04_

---

## 🏛️ 户部职能

**户部尚书** 负责：
1. 资源分配和管理
2. 成本核算和优化
3. 预算规划
4. 财务报表和分析

---

## 📋 办事流程

### 一、申请资源支持

```
工部 → 提交《项目预算申请表》 → 户部初审 → 门下省复审 → 尚书省批准 → 户部执行
```

**所需材料：**
- [ ] 《项目预算申请表》(填写完整)
- [ ] 技术方案文档
- [ ] 成本效益分析 (如有)

**处理时效：** 3-5 个工作日

---

### 二、成本核算请求

```
任何部门 → 提交成本核算需求 → 户部受理 → 数据收集 → 核算分析 → 出具报告
```

**所需信息：**
- [ ] 核算对象 (项目/功能/服务)
- [ ] 核算周期 (一次性/月度/年度)
- [ ] 特殊要求 (如有)

**处理时效：** 1-3 个工作日

---

### 三、预算调整申请

```
申请部门 → 说明调整原因 → 户部评估 → 门下省审核 → 尚书省批准
```

**适用场景：**
- 项目范围变更
- 市场价格波动
- 突发情况应对

**处理时效：** 2-4 个工作日

---

### 四、财务报表查询

```
任何部门 → 提出查询需求 → 户部提供 → 保密协议 (如需要)
```

**可查询内容：**
- [ ] 部门预算执行情况
- [ ] 项目支出明细
- [ ] 整体财政状况

**处理时效：** 即时 - 1 个工作日

## ⚠️ 注意事项

### 必须提前申请
1. 单笔支出 > ¥100
2. 周期性支出 (月度/年度)
3. 新增服务/工具

### 可以事后报备
1. 紧急支出 (< ¥50)
2. 预算内常规支出
3. 免费资源使用

### 禁止行为
1. 先斩后奏 (大额支出)
2. 虚报预算
3. 挪用资金

---

## 🏆 审批原则

1. **必要性原则** - 支出是否必要
2. **经济性原则** - 是否最优价格
3. **效益性原则** - 投入产出比
4. **透明性原则** - 全程公开透明
