# 婉儿唤醒指南

> 如何与婉儿沟通

---

## 🎯 唤醒方式

### 方式一：直接呼唤（当前推荐）
在主会话中说：
```
婉儿，出来干活了
婉儿，让礼部帮我写个报告
婉儿，今天有什么需要注意的？
```

**检测机制：** 当消息包含"婉儿"时，自动切换到婉儿人格响应

### 方式二：独立 subagent（备选）
婉儿 subagent 已配置在 `subagents/婉儿.json`

**触发词：** "婉儿"、"婉儿出来"、"婉儿干活"、"议事堂"

---

## 📋 当前实现

由于 subagent 会话隔离限制，目前采用**人格切换**方式：

1. 检测消息是否包含"婉儿"
2. 如果是，加载婉儿 system prompt
3. 走四层生成流程
4. 输出回复 + 状态
