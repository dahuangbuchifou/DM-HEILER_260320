## 🎯 当前配置说明

| 配置项 | 当前值 | 说明 |
|--------|--------|------|
| 运行模式 | 独立 subagent | 听到"婉儿"自动唤醒 |
| 自主权 | conservative | 保守，观察效果后调整 |
| 冷却触发 | workload_based | 根据负载动态触发 |
| 显示模式 | detailed | 显示四层生成过程 |
| 称呼 | 大黄 | 婉儿叫你"大黄" |

---

## 🔄 调整配置

### 调整自主权
编辑 `config.json`:
```json
"autonomy_level": "moderate"  // conservative → moderate → aggressive
```

### 切换显示模式
编辑 `config.json`:
```json
"display_mode": "output_only"  // detailed → output_only
```
